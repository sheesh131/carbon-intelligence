1️⃣ Introduction  
Modern machine learning models increasingly demand high computational resources, leading to 
significant energy consumption and carbon emissions. Traditional model development focuses 
primaril y on predictive performance while ignoring environmental impact.  
This work introduces a carbon -aware neural architecture optimization framework that integrates 
sustainability directly into model design. Instead of optimizing accuracy alone, the proposed sy stem 
jointly considers:  
 predictive performance,  
 inference efficiency,  
 computational carbon cost.  
The framework combines neural architecture search (NAS), dynamic model scaling, and adaptive 
numerical precision to discover sustainable credit risk models.  
 
2️⃣ Objectives  
The primary research objectives were:  
1. Design a carbon -aware NAS system incorporating carbon cost as an explicit optimization 
objective.  
2. Enable dynamic scaling of model architecture during inference.  
3. Support adaptive precision execution (FP32 , FP16, INT8).  
4. Evaluate sustainability -performance tradeoffs in credit risk modeling.  
5. Demonstrate reproducibility and generalization across datasets.  
 
3️⃣ Methodology  
 
3️.1️ Carbon -Aware Neural Architecture Search  
The NAS system evaluates candidate architectures using a multi -objective formulation:  
Objective = maximize(AUC)  
            + minimize(carbon cost)  
            + maintain decision stability.  
Search dimensions included:  
 hidden layer scaling (0.5, 0.75, 1.0)  
 precision modes (FP32, FP16, INT8)  
 adaptive early -exit levels.  
 
3️.2️ Carbon Cost Modeling  
Carbon cost was estimated using proxy metrics:  
 preprocessing latency  
 inference latency  
 precision -dependent scaling factors.  
This approach enables carbon -aware evaluation without requiring external ene rgy measurement 
hardware.  
 
3️.3️ Dynamic Scaling and Adaptive Precision  
The system dynamically selects:  
 reduced model width (hidden_scale)  
 numerical precision level.  
This allows balancing performance and energy usage.  
 
4️⃣ Experimental Setup  
 
Datasets  
1️. Bank Dataset  
Used for multi -seed robustness experiments.  
2️. German Credit Risk Dataset  
Used to validate cross -dataset generalization.  
 
Evaluation Metrics  
 AUC (Area Under ROC Curve)  
 KS statistic  
 Brier score  
 Carbon cost proxy  
 Efficiency = AUC / carbon.  
 
Experimental Protocol  
 Multi -seed evaluation (5 seeds).  
 Architecture retraining per configuration.  
 Pareto frontier analysis.
